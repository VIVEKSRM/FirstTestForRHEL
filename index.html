This is my page

This page is loaded using a nginx server

nginx server is inside a docker container

Docker container is created via Jenkins

Jenkins will act as a centralized location for all the integration

The flow is

When we do some changes and commit the changes in the Git Reprository
Jenkins jobs will get triggered automatically after the commit
Jenkins first create an NGINX docker image with the current files
Jenkins then delete the old docker container and create a new docker container with the udpated files
